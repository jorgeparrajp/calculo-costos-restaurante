Sistema de calculo de costos de platos en un restaurante

Proyecto en lenguaje C para gestionar ingredientes, platos y relaciones plato-ingrediente.
Calcula costo base, impuesto, servicio, ganancia y costo final.
La informacion se guarda en archivos CSV.

Compilar:
gcc main.c restaurante.c -o restaurante.exe

Ejecutar:
./restaurante.exe
